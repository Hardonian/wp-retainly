# WP Retainly

Predict & reduce churn in WooCommerce shops using AI-driven behavior signals.

## Features
- Churn risk scoring
- Behavioral event tracking
- Webhook notifications
- Admin dashboard
- WooCommerce integration

## Installation
1. Upload to /wp-content/plugins/
2. Activate plugin
3. Configure settings

## Requirements
- WordPress 6.0+
- WooCommerce 7.0+
- PHP 8.0+
